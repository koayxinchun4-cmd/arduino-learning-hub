## How To Use / 使用方式

Follow the folders in order. Each builds on the previous.  
照著資料夾順序走，每一個都建立在上一個的基礎上。

| # | Folder | Topic | Time |
|---|---|---|---|
| 01 | 01-getting-started/ | Zero to One — 零基礎入門 | ~3 hours |
| 02 | 02-wokwi-projects/ | 5 Wokwi 實作專案 | ~5 hours |
| 03 | 03-code-examples/ | 程式碼範例庫（隨時參考） | Reference |
| 04 | 04-troubleshooting/ | 疑難排解 | When stuck |
| 05 | 05-advanced/ | 進階：中斷、WiFi IoT、綜合專案 | ~8 hours |

---

## Tools You Need / 工具

| Tool | What It Does | Cost |
|---|---|---|
| [Wokwi](https://wokwi.com) | Browser Arduino/ESP32 simulator | Free |
| [Velxio](https://velxio.dev) | Alternative simulator — 19 boards, browser-based | Free |
| [Arduino CLI](https://github.com/arduino/arduino-cli) | Command-line Arduino tool (for real hardware later) | Free |

Both Wokwi and Velxio run entirely in your browser. No hardware required to start.

Wokwi 和 Velxio 都在瀏覽器裡跑，完全不需要買硬體就能開始。
